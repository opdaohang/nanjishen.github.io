7-zip 命令行的使用，先将 7-zip 加到系统的环境变量里


解压文件：7z x {fileName} -o{outputDirectory}


# 解压 C 盘下的 1.zip 到 C 盘下的 test 目录里（注意：o后面没有空格）
7z x C:\1.zip -oC:\test

# 解压当前目录下的 1.zip 到当前目录下的 test 目录里
7z x 1.zip -otest

添加文件：7z a


# 添加当前目录下所有的文件到 1.7z 的压缩包（没有则创建）
7z a 1.7z

# 添加当前目录下 test 目录里的所有文件到 1.7z 压缩包里
7z a 1.7z test/

删除文件


# 删除 1.zip 压缩包中的所有扩展名为 bak 的文件：
7z d 1.zip *.bak -r

命令行里输入 7z.exe 可以查看其它可用操作：


  • a：将文件添加到压缩档案中
  • b：测试压缩或解压算法执行时的 CPU 占用
  • d：从压缩档案中删除文件
  • e：将压缩档案中的所有文件解压到指定路径，所有文件将输出到同一个目录中
  • h：计算文件的哈希值
  • i：显示有关支持格式的信息
  • l：列出压缩档案的内容
  • rn：重命名压缩档案中的文件
  • t：测试压缩档案的完整性
  • u：更新要进入压缩档案中的文件
  • x：将压缩档案中的所有文件解压到指定路径，并包含所有文件的完整路径

    Everything


    字符作用示例
    * 通配符*.pdf（任何名后缀位pdf文件）
    space 与01 ok（包含01与ok）
    | 或123|456（包含123或456的文件）
    dm: 日期*jpg dm:2016/10/10-:2016/10/20（2012年10月10日到20日的.jpg文件）
    X:\XX 路径D:\folder 123（D盘folder文件夹下包含123的文件）
    size: 大小*.zip size:>300MB（所有大于300MB的.zip文件）
    ? 通配符???.txt（xxx.txt文件（和*的区别是一个?号代替1个字母）
    . 任意单字符nj.hen（nj任一字符hen，结果njshen）
    ^ 头部^game（以game开头的）
    $ 尾部game$（以game结尾的）
    [] 包括字符[abc]（a 或 b 或 c）
    [^] 不包括字符[^abc]（不包括a 或 b 或 c）
    [-] 任意字符[a-z]（从 a 到 z 之间的任何一个字符）
    [--] 任意字符[a-zA-Z]（从 a 到 z, 及从 A 到 Z 之间的任何一个字符）

    DokuWiki


    安装配置


    运行安装：浏览器访问 dokuwiki/install.php
    中国时区：conf 目录下添加 local.protected.php 文件，写入如下内容： 
    <?php>date_default_timezone_set("Asia/Chongqing");?>

    刷新页面缓存


    在页面的url后加&purge=true
    导出文件：在页面url后加&do=export_xhtml（支持raw、xhtml、xhtmlbody）
    升级备份等需要备份此三个文件：
  • 配置：conf/local.php
  • 权限：conf/acl.auth.php
  • 用户：conf/users.auth.php
    • （如果，将文件名保存成类似conf/local.protected.php每次在在管理后台保存配置时，就不会覆盖 *.protected.php 中的配置）

    配置设置：

  • showuseras 显示用户为：建议选用户全名，这就不会泄露用户登录帐号
  • gzip_output 对 xhtml 使用 gzip 内容编码：开启
  • send404 发送 "HTTP 404/页面没有找到" 错误信息给不存在的页面：开启
  • dnslookups 查询主机名：建议禁用

    • 侧边栏配置

  • 配置设置里sidebar项写侧边栏页面路径和名称（默认sidebar）
  • 创建这个页面（比如建立一个文本文件名为sidebar，侧边栏就出来了）
    • （当同级命名空间下有相应的侧边栏页面时，则使用此页面）


    推荐插件

  • IndexMenu Plugin：侧边栏索引导航，下载：https://www.dokuwiki.org/plugin%3Aindexmenu>
  • Add New Page Plugin：新建页面，下载：https://www.dokuwiki.org/plugin:addnewpage
  • Move Plugin插件： 移动页面和命名空间，下载：https://www.dokuwiki.org/plugin:move

    • 备份方法：（DokuWiki不使用数据库，所以可以备份整个根目录）

    • data/pages：页面文档目录（主要备份）
    • data/meta：页面描述档（如谁创建，谁修订）
    • data/media：多媒体档 （图片，PDF等）
    • data/media_meta：多媒体描述档
    • data/attic：旧版本页面
    • data/media_attic：旧版本媒体档
    • conf：设定档
    • lib/plugins：插件目录（备份你要的）
    • Lib/tpl：模板目录

    （可以忽略：cache，index，locks， tmp文件夹）
